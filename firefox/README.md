# My firefox configs.

## How to use.
Place in: `~/.mozilla/firefox/<profilestring>.default/`
  
You must restart firefox for the CSS to reload!

### For example: 
`~/.mozilla/firefox/<profilestring>.default/chrome/userChrome.css`
